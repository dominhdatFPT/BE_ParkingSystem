II. FLOW CHỨC NĂNG TRONG MÀN HÌNH TỔNG QUAN (STAFF)
1. Truy cập trang Tổng quan
   Thay vì yêu cầu Admin Code, hệ thống chỉ cần kiểm tra tài khoản đã đăng nhập có quyền Staff trở lên hay chưa.
   Mở website → Truy cập trang Tổng quan → Hệ thống kiểm tra phiên đăng nhập và phân quyền Staff → Hệ thống tải giao diện → Truy cập thẳng vào màn hình Dashboard.
